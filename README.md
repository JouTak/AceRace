# AceRace - гоночный режим
AceRace - режим, где игроки (от 1) должны пройти трассу на скорость.

## Установка
1. Установи **Multiverse-Core**
2. Скопируй `AceRace-<version>.jar` в `plugins/`
3. Подготовь миры:
  - лобби-мир (например lobby),
  - template-world для каждой трассы (например AceRaceTemplate) - миры должны быть загружены на сервере через MultiVerse
4. Запусти сервер - плагин создаст файлы:
  - `plugins/AceRace/config.yml`
  - `plugins/AceRace/checkpoints.yml`
  - `plugins/AceRace/zones.yml`
5. Настрой `config.yml`, `checkpoints.yml` и `zones.yml`, затем перезапусти сервер.

#### Заполнение чекпоинтов через игру:
1. Перейти в **мир-шаблон** через команду `/mvtp <worldname>`
2. Встать в первую точку чекпоинта и ввести команду `/cp setpos1 <index>`, где index - номер чекпоинта (отсчет начинается с 0)
3. Встать во вторую точку чекпоинта и ввести команду `/cp setpos2 <index>`
4. Проделать эти действия для всех чекпоинтов
5. Ввести команду `/cp save`, чтобы сохранить все чекпоинты в конфиг-файл.

#### Примечание
  - Советую делать чекпоинты хотя бы 2 блока толщиной, чтобы все точно работало правильно и чекпоинт не пропускался из-за большой скорости.

#### Способ заполнения через игру

  1. Перейти в **мир-шаблон** через команду `/mvtp <worldname>`
  2. Встать в первую точку зоны и ввести команду `/zone setpos1 <name> <type>`, где type - тип зоны `BARRIER/UNDERWATER/ELYTRA`
  3. Встать во вторую точку чекпоинта и ввести команду `/zone setpos2 <name> <type>`
  4. Проделать эти действия для всех зон
  5. Ввести команду `/zone save`, чтобы сохранить все зоны в конфиг-файл.


## Команды
### Для чекпоинтов
- `/cp setpos1 <index>` - первый угол зоны
- `/cp setpos2 <index>` - второй угол зоны, сохранение зоны
- `/cp remove <index>` - удаление все зоны с индексом
- `/cp list` - список всех чекпоинтов
- `/cp save` - сохранить все чекпоинты этого мира в конфиг
- `/cp reload` - перезагрузить зоны из файла
- `/cp clear` - удалить все чекпоинты
- `/cp info` - показать maxCheckPointIndex и кол-во чекпоинтов
- `/cp worlds` - показать все миры с чекпоинтами
- `/cp test` - проверить стоит ли игрок в зоне какого-либо чекпоинта

### Для зон
- `/zone setpos1 <name> <type>` - установить первую точку
- `/zone setpos2 <name> <type>` - установить вторую точку
- `/zone remove <name>` - удалить зону
- `/zone list` - список всех зон
- `/zone save` - сохранить все зоны в конфиг
- `/zone reload` - перезагрузить зоны из конфига
- `/zone clear` - удалить все зоны
- `/zone info <name>` - инфа о зоне
- `/zone test <name>` - проверить внутри ли игрок


## Чеклист
1. Запусти сервер и проверь, создались ли `config.yml`, `checkpoints.yml`, `zones.yml`
2. Заполни чекпоинты и зоны одним из двух способов
3. Убедись, что мир шаблона загружен на сервер
4. Зайди на сервер и пропиши `/ar ready`
5. При старте гонки должны быть барьеры, ограничивающие ходьбу по карте до начала гонки
6. Попробуй пройти все чекпоинты и завершить гонку, должно работать без перебоя. Если чекпоинт пропускается, попробуй увеличить его размер в толщину на 1.
